Behind the Scenes

What makes our model function?

VGG19

VGG19 is 19 layer deep convolutional neural network, created by the Visual Geometry Group of Oxford University in 2014. It is often used to classify images, but we used VGG19 for creating the style transfer.

Python (Flask)

We used Flask to create this webapp. Flask was released in 2010 by Armin Ronacher.

Bootstrap

Bootstrap has many templates for various websites. Our website uses the "Ninestars" Bootstrap template.

AI Camp

AI Camp, founded by Michael Zhang, creates summer programs for teens to learn about AI. Through AI Camp, this team was able to meet each other and create this project.
